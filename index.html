霍尔效应

发表于 2022-08-25 更新于 2022-08-26

中科大的侯老师给我布置了这几个任务：1. 什么是spin Hall effect
2. 光子为什么会有spin Hall effect，和电子的spin Hall effect有什么不同
3. 中科大这些同学的实验和文献中的结果相比，哪个信噪比更好？如果定量评价实验的信噪比？
同时给了我几篇参考文献，进行学习，虽然我不去中科大了，但是这个任务我还是决定做一做，毕竟假期闲来无事。
首先需要解释的就是：1、霍尔效应，当电流垂直于外磁场通过半导体时，载流子发生偏转，半导体两次出现电势差，称为
霍尔电势差。
2、反常的霍尔效应则没有外加的磁场，由材料自身的磁化而发生的，当电流通过铁磁材料时，费米面附近自旋向上和自旋向下的电子数目不相等，自旋向上与自旋向下的电子由于受到自旋相关散射和自旋-轨道耦合的作用，将向相反的方向偏转，导致在两端的界面处出现自旋极化现象和电势差。
3、自旋霍尔效应（spin Hall effect），主要出现在重金属材料中，与反常霍尔效应类似，自旋向上和自旋向下的电子受到自旋轨道耦合等效场的作用，导致了自旋极化，不用自旋的电子在材料两端堆积，但是不存在电势差，两侧电子数量一致，存在自旋流。（自旋流即设自旋朝上的电子和自旋朝下的电子以相同的平均速度反向运动，两种流的绝对值相等，方向相反，因而没有净电荷的流动, 只有自旋的流动。）
那么光子为什么会存在自旋霍尔效应呢？首先，我们将左旋光和右旋光看成电子自旋向上和向下的两种态。紧接着类比折射率梯度为外加电场的角色。光子有自旋，但是为中性粒子，无磁矩，因此无法用外加场的方法改变其自旋轴的方向，但是光子自旋轴的方向与传播方向一致，我们可以利用光的折射与反射，其本质是设置一系列折射率缓慢变大的材料（折射率梯度）。最终左旋分量与右旋分量在垂直于入射面的横向产生一个很小的相对位移。（为了满足角动量守恒，而产生的微小横移）具体理论计算，打字表示较复杂，这里不给出了。（利用平面角谱理论结合光的传播定律推导）
平面角谱理论：把孔径平面光场分布看作沿不同方向传播的平面波分量的线性组合。（光在非均匀折射率中移动，不同的角谱分量相位改变不同，不同角谱分量相互干涉的结果是产生两束分裂的光束。）
下面介绍一下这个实验：《空气-玻璃表面反射光光子自旋霍尔效应 》，这个实验的目的是测量出左旋光和右旋光确实分裂了，但是由于分裂非常微小，需要利用弱测量放大的方法增加分裂进行测量。格兰泰勒棱镜（激光偏振器）：输入一束无偏光的光束，可以得到一束线偏振光。利用两个格兰棱镜，第二个格兰激光偏振镜P2 与第一个格兰激光偏振镜P1接近垂直,但是偏移一个小的角度𝛥。这个角度与放大系数有着密切联系，如果粗略考虑那么，这个角度越小，放大倍数越大，但实际情况是当角度趋近于0时放大倍数也会趋近于0，观察到的分裂为0。但当角度很大时，此时弱测量机制影响就很小了，考虑到入射光为高斯光束，此时的放大主要由传播放大所贡献，因此要控制这个角度的大小。在测量时，选取不同的入射角度，抓取图片，以同组图片中|𝛥|最大的两张图片的光束质心位置平均值作为偏移为 0 的位置，分别计算同组不同𝛥值的质心偏移量，得到同入射角θ在不同𝛥下的偏移量，再与弱测量放大修正公式作图比较。
最后对于定量评价信噪比，我认为只需要利用实验数据与公式求出的标准数据做差后除以标准数据乘以100%。科大同学和文献中哪个信噪比更好，只需要代入数据简单计算对比。

　　而老一辈人则不然。受到封建思想熏陶的那一辈人必定是认为男性的主导地位不可动摇，甚至更为夸张的是，同一辈中女性甚至比男性更加支持男权。最简单的例子就是农村重男轻女的反而女性占比更高。老一辈中认为女性赚钱多，就是男的吃软饭，结婚双方，女生长得不能比男生高，等等。
五、第四章—女性篇，男权的产生
　　以上论述了男性的一生四个阶段，女性也存在四个阶段，但由于我是男性，没有过多的实践能力，我大致把女性的四个阶段划分为：男色，外貌，金钱，安稳。
　　其中男色和外貌大致是在同一时期追求的，但由于学校等外在因素的限制，对于男色的追求要略早于对自己外貌的追求。哪个少女不思春。
　　处于第一阶段的女生，大致处于初中至高中，（女性的思想进程前期早于男性后期落后于男性）她们会或疯狂或有限度的迷恋上一个帅哥明星，仅仅是因为外貌符合她们的审美，有些人不惜为自己的爱豆付出所有，有很多人花费父母的积蓄给素未谋面的明星打榜。她们也不知道自己到底喜欢明星的什么地方。再后来她们所追求的便是现实世界中的白马王子，希望能找到一个人当自己的男朋友。在这个时期，我可以负责任的讲，普通初中高中的普通男生，他们的行为品德或好或坏，但本质都是纯真的，思想达不到一个高度，不会产生将女人当玩具戏弄的想法。所以男权在这一阶段不可能存在。
　　而处于第二阶段的女生，开始了对于外貌的追求。女生追求外貌往往是与追求金钱挂钩的，前后相差不了几年。不要说什么打扮打扮不是为了给别人看。打扮的漂亮的目的就是为了吸引外人的注意力，女性本就容易缺乏安全感，渴望获得更多人的注意。况且，这个社会对于外貌漂亮的女生存在有太多太多不合理的特权了。追求外貌对于女性来说是一个提升自我价值的好方式。
但是不要走上另一个极端，靠自己的外貌来赚钱，许多年轻女性会陪着一位中年大叔。你要知道，中年大叔的阅历远高于你，他在商界或政坛打拼多年，他深知你是一个贬值的商品，你的美貌每年都会降低，没有人会购买你，最好的办法是租赁。你把他们当提款机，他把你当成玩具。可以说这是各取所得，但是没有人会永远年轻，但总有人年轻，老了以后，漂亮女生的剩余价值就很有限了。
我看到的是，许多漂亮女生，还是一直非常努力的提升自己的思想，不断学习各种的知识，所以上面提到的只是阴暗面的例子。
　　所以到现在为止，还没有引出男权的产生。其实我也很疑惑，为什么会存在极端的男权。我目前有两种猜测，这里只给出一种：
　　首先男性对于权力的向往要远高于女性，这并不只有中国，可以从任何一个国家的最高权力机关中寻找。2020年，女性在全球议员中所占的比例超过了25%，创下历史新高。在我看来，这个比例永远不可能持平，根源在于男女双方的激素影响不同，天性并不同。这就好比计算医院中护士的男女比例一样，毫无意义是女生要多。而普通的男性并没有办法在政坛叱咤风云，只能通过在家庭中谋取地位，来满足自己对于权力的欲望。前面提到过，四个阶段，会同时出现，但是重心的顺序不会变，他们重心依然在金钱，但是对于权力也存在着欲望，为了达到这种欲望的满足，就造成了男权的产生。
　　女性对于平权的抵抗感可能来自于女性客观上的社会弱势地位，这种弱势并不是平权所能解决的。还有重要的一点就是对于几千年来男权社会的一种反抗心理。
六、第五章—总结
　　首先第一句话：这种话题根本没啥好聊的，只要聊的人不是一个性别，就鲜有意见相同的时候。
　　当然以上这些原因并不健全，但是足以论证我的观点：男女双方都有足够有力的证据来保全自己的立场，男女权不可调和的根本原因是立场的不同。但是归根到底，与男权女权争论一些问题是相当的愚蠢的，我也犯过类似的错误。他们不会和你盘逻辑，只认为自己的想法无比正确。不如多读书，思想的进步会更快。
　　最后整点鸡汤，努力总是没错的，争取自己的利益不能通过贬低对方，你若盛开蝴蝶自来。千帆过尽，希望大家认清生活的真相后依然热爱生活，并找到真正属于自己的另一半。

三观

发表于 2022-02-14

　　要谈三观，首先要明确什么是三观，三观即:世界观 人生观 价值观。世界观是人类对整个世界的总的看法和根本观点。人生观就是对于人类生存的目的、价值和意义的根本看法和态度。价值观是指一个人对周围的客观事物的意义,重要性的总评价和总看法。
　　世界观、人生观和价值观三者是有机统一的：有什么样的世界观就有什么样的人生观，有什么样的人生观就有什么样的价值观。
　　人们常说的三观不合便是这三观不合。有过民间调查，有百分之64的人表示最看重自己与对象的三观是否相符，朋友之间也不例外，
　　我们通常所说的：尿不到一个壶里去。三观不一致的人，在一起相处，无论如何也走不长远。下面我将以自己的例子浅显的讲解一下影响三观的因素。下面列举的三个因素是由影响力大到影响力小排序的。
一：自己的觉悟
　　自身觉悟的影响应该是最大的，或许有那种一瞬间突然就想通了，突然就觉悟了。可能是读了一本书，可能是经历了一些事，三观便被自己内化的改造了，并不是被外界直接影响的。区别在于有了自己的思考，可以不断更新自己的观点，并对于外界的观点拥有区分的能力。
二：社会环境因素
　　社会环境因素对于年经人（未完成自我认知的人统称为幼稚的年轻人）的影响是致命的。在三观尚未完全成型的时期，社会的环境在这四个因素中占据主导因素，包括现在大环境下造就的某些极端爱国者，极端女权，极端男权等。注意我加了极端两个字，爱国没有问题，身为女性/男性，为自己的性别争取一些优势也说得过去。但是这些都建立在有理可依的基础上，极端的人在面对你抛出的观点后，不会与你据理力争，不是所谓的诤友，而像是一条疯狗一样，扑在你身上进行撕咬……经历过的会懂那种感受……
　　当前的社会，铺天盖地的主旋律下，是否隐藏着危机？极端男权女权红小将，甚至多种极端融为一身，是否隐藏着危机？具体不再论述。
三：教育
教育通常被视为是三观形成的重中之重，为什么我将其放在了第三位呢？
　　只有经历了否定之否定，才能认识到自己的问题。例如：我现在回首高中的教育，我总觉得老师紧紧是一个引路人，不必任何事情都跟着老师，师傅领进门，修行在个人。但是在当时那个大环境下，几乎没有人敢违背老师的安排。为什么呢？一定是害怕老师批评，更害怕自己掉队。虽然现在我认识到了，当时自己自学或许取得的成绩要比现在好很多，但是未来的我，并不会给自己孩子说这些。我现阶段认为：每个人的路都要一步步走，必须经历多次的自我否定。我如果直接告诉他不要听老师的话&&&，他并不能理解&&&的意思，只能记住前面的“不要听老师的话”。这并不是我想看到的。
四：朋友
　　物以类聚，人以群分，这句话我是深信不疑的。你在什么圈子，接触什么人，很大程度上会影响你的三观。但是我为什么把朋友放在了最后一位，是因为我认为朋友之间的三观本就是相互影响相互渗透的，并不是一味的接收或者一味发送的。而且在拥有独立思考的能力之后，会看清有些朋友可以继续，有些朋友只能渐行渐远。

我觉得随着我的人生不断的进步，我对于自己三观的认知也会不断的刷新。
回过头来看看当初幼稚的自己，才会发觉自己的成长。不要轻易的相信别人的言论和判断，保持一颗谦虚且独立的内心。

2022你好

发表于 2022-02-08

转眼间，2022年了，我也是大三下的老油条了，应该计划一下未来了。
感觉自己大学什么都学了，又什么都没学会，身边的朋友有很多人也是同样的迷茫。
回首2021年，迷茫与天赋，是我emo时，提及最多的两个词。
抬头向上看，无数的强者还在我的头上。刚上大学时，以为自己的天赋已经快要登上金字塔的顶端了，现在看来，
自己的天赋只不过是金字塔底部的一只小蚂蚁。这个世界太大了，太参差了。但不管怎样，生活还是要继续。

不那么孝顺了。
2022开始了，今年就先克服知道与做到之间的鸿沟吧，新的一年，多读书，好好生活。祝福大家万事顺心。
